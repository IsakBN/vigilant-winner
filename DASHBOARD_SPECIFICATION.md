## Overview

BundleNudge is an OTA (Over-The-Air) update platform for React Native applications. The dashboard allows developers to:

- Push JavaScript updates without App Store review
- Manage apps, releases, and channels
- Monitor device adoption and rollout progress
- Configure A/B tests and staged rollouts
- Manage teams and collaborate

### Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | Next.js 15 (App Router) |
| React | React 19 |
| Styling | Tailwind CSS |
| Components | shadcn/ui |
| Data Fetching | TanStack Query |
